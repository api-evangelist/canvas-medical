Canvas Medical introduced the Canvas Companion, a mobile EMR platform letting providers access patient records and documentation on smartphones. It uses a plugin architecture across three tiers (system-wide applications, patient-specific tools, and encounter-level documentation), with all functionality delivered through plugins built on the Canvas SDK.
