Canvas Medical launched Navigator, an AI agent that answers customer questions in Slack by drawing on Canvas documentation, SDK materials, and customer-specific deployed plugin code to give instance-specific guidance. Evolved from the internal Investigator agent, it can answer questions like which FHIR resource to use for eligibility checks; it is in beta with general availability expected summer 2026.
