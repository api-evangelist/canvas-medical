Canvas Medical launched Canvas Studio, a no-code workflow builder powered by Claude Code that lets clinicians and staff create custom EMR plugins using natural language. The company says work that once required a multi-year project with engineers, product managers, and designers can now be completed by clinicians in a few hours.
