# Canvas Medical

Canvas Medical is a developer-first EHR platform built for virtual and value-based care organizations. The platform provides a comprehensive FHIR R4 REST API with 41 resources and a native Python SDK for building custom clinical workflows, scheduling, charting, billing integrations, and care team coordination tools.

## APIs

This repository tracks two primary developer-facing APIs:

### Canvas Medical FHIR API
A FHIR R4-compliant REST API providing access to 41 FHIR resources for clinical, administrative, financial, and care coordination workflows. Secured with OAuth 2.0 and SMART on FHIR scopes.

### Canvas Medical SDK
A Python SDK available on PyPI (`canvas`) enabling developers to build plugins that execute natively within the Canvas EMR infrastructure, responding to 650+ clinical and operational events.
